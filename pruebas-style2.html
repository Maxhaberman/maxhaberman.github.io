Ejemplo de Box Model

Este párrafo está dentro de un contenedor con un borde y relleno. El box-sizing está configurado en "border-box", lo que significa que el ancho del contenedor incluye el borde y el relleno.

El modelo de caja es un concepto fundamental en CSS que define cómo se calculan las dimensiones y el espacio alrededor de los elementos HTML. Es importante comprender el box model para crear diseños web consistentes y receptivos.
